# movie-information-catalog-record
本仓库用于整理电影与电视剧相关网站的公开信息，包括站点名称、页面关键词布局以及内容更新节奏等。通过归档式记录方式，对影视信息平台进行系统性整理，便于长期观察与比对分析。项目不涉及资源转载或下载，仅记录公开页面所展示的基础信息，作为资料型笔记存在。
